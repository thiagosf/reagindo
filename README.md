# Reagindo

Ambiente de desenvolvimento de aplicativos __React__.

## Instalação

Baixar o repositório:

```git clone https://github.com/thiagosf/reagindo.git my-awesome-project```

Instalar dependências:

```npm install```

Executar o webserver:

```gulp webserver```
